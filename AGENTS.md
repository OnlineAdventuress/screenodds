# ScreenOdds Project Instructions

## Project

ScreenOdds is a Next.js SEO/data site for entertainment prediction markets at `screenodds.com`.
The first verticals are movies, box office, awards, TV/streaming, and reality TV.

## Critical Rules

- Keep this as a separate project from `aimodelodds`; do not overwrite or rebrand that repo.
- Do not hardcode API keys. Load local keys from `C:\Users\longl\Desktop\Shared-Sync\.env` and keep project `.env.local` gitignored.
- Polymarket data must render with deterministic fallback fixtures so the site builds when upstream APIs fail.
- US users may be restricted from Polymarket trading. Do not present trading CTAs to blocked users without a geoblock check.
- Affiliate/referral links must have visible disclosure and `rel="sponsored nofollow"` where appropriate.
- SEO pages must be useful data pages, not thin affiliate pages. Each article must add original context, live market data, internal links, and source-backed explanations.
- Kie.ai / ChatGPT Image 2 assets must be generated by script into `public/images/` or `public/blog/` and reviewed before use.
- Avoid generic `movie odds` targeting; DataForSEO shows that query is polluted by the film title *Against All Odds*.

## Tech Stack

- Framework: Next.js with App Router
- Language: TypeScript
- Styling: Tailwind CSS
- Tests: Vitest
- Data: Polymarket Gamma/CLOB APIs with local fixtures
- Market discovery: Parlay API, optional and server-side only, with deterministic fallbacks
- Social sentiment: xAI/Grok X Search is optional and must run from server-side scripts only
- Keyword research: DataForSEO API
- Images: Kie.ai ChatGPT Image 2 API
- Hosting target: Netlify for `screenodds.com`

## SEO Rules

- Every indexable page needs a unique title, description, canonical URL, H1, crawlable internal links, and useful visible text.
- Use `Article`, `FAQPage`, `BreadcrumbList`, and `WebSite` JSON-LD only where accurate.
- Keep hub pages within three clicks of the homepage.
- Every market/detail/article page links back to its hub and to 2-5 related pages.
- Do not keyword-stuff. Match natural search intent.
